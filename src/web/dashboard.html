Auto-Update Server

                        
When enabled, oxi-dns automatically checks for updates every 8 hours. New
                            versions are health-checked before being applied. If a health check fails, the
                            current version continues running and a notification is shown.

                    

                    
                

                

                    

                        
IPv6 (AAAA)

                        
Include IPv6 records in DNS responses. Disable on IPv4-only networks to prevent
                            clients from attempting unreachable IPv6 connections.
